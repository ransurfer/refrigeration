Don’t get caught out by the R22 Phase Out!

Since the 1st January 2015, it has been illegal to top up an HCFC system. This means that if you own R22 equipment, it cannot be repaired if it begins to leak. This makes it impossible to repair and difficult to maintain.

Any equipment that was purchased before 2004, is better off being replaced. Firstly because it will be energy efficient and therefore cost less to run and secondly, because you will be able to maintain the system and make sure that it continues to run as it should.
